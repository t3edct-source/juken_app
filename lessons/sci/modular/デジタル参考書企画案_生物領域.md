# 小学校中学受験理科 生物領域 デジタル参考書 企画案

## 📱 コンセプト

**「いつでも、どこでも、楽しく学べる生物の世界」**

スマホ・タブレットに最適化された、視覚的でインタラクティブな生物参考書。既存の24レッスン、300+画像を活用し、中学受験に必要な生物知識を体系的に学習できる。

---

## 🎯 ターゲット

- **主対象**: 小学校4-6年生（中学受験準備）
- **学習環境**: 通学中、自宅、外出先など
- **デバイス**: スマートフォン（縦向き最適化）、タブレット（横向き・縦向き対応）

## 🎨 UI/UX設計

### 2. **参考書モードの設計**

#### 2-1. ページ構成
```
┌─────────────────────────┐
│  [←戻る]  植物の観察  [🔍] │ ← ヘッダー（固定）
├─────────────────────────┤
│                         │
│   📷 大きな図解画像      │ ← 画像（タップで拡大）
│                         │
│   ## 花のつくり          │ ← 見出し
│                         │
│   花は植物が子孫を残す   │ ← 本文（読みやすいフォント）
│   ための特別な器官です。 │
│                         │
│   💡 ポイント            │ ← 重要ポイント（ハイライト）
│   ・めしべとおしべの役割│
│   ・受粉の仕組み        │
│                         │
│   ┌─────────────────┐  │
│   │ 関連問題に挑戦   │  │ ← CTAボタン
│   └─────────────────┘  │
│                         │
│   [前へ]  [目次]  [次へ] │ ← ナビゲーション
└─────────────────────────┘
```

#### 2-2. スマホ最適化のポイント
- **縦スクロール**: 1画面に1テーマ、スムーズにスクロール
- **画像拡大**: タップでフルスクリーン表示、ピンチズーム対応
- **フォントサイズ**: 16-18px（読みやすさ重視）
- **余白**: 十分な余白で視認性向上
- **タップ領域**: ボタンは44px×44px以上

#### 2-3. タブレット最適化のポイント
- **2カラムレイアウト**: 横向きで画像と本文を並列表示
- **横スクロール**: 図解を横にスワイプで閲覧
- **マルチタッチ**: 複数画像を同時に拡大

## 🎮 インタラクティブ機能

### 3. **学習を深める機能**

#### 3-1. 画像インタラクション
- **ホットスポット**: 図解の部位をタップすると説明が表示
  ```
  例: 花の図解
  - めしべをタップ → 「めしべは種子を作る部分です」
  - おしべをタップ → 「おしべは花粉を作る部分です」
  ```

- **アニメーション**: プロセスを動画で説明
  ```
  例: 消化の流れ
  - 食べ物が口→食道→胃→小腸→大腸と流れるアニメーション
  ```

- **比較表示**: スワイプで比較
  ```
  例: 春の植物 vs 夏の植物
  - 左右にスワイプで切り替え
  ```

#### 3-2. 問題演習機能
- **即座フィードバック**: 選択後すぐに正誤表示
- **解説表示**: 間違えた問題は自動で解説ページへ
- **進捗バー**: 学習進捗を視覚化
- **タイマー**: 制限時間モード（本番対策）

#### 3-3. 学習支援機能
- **ブックマーク**: 重要ページを保存
- **メモ機能**: ページに手書きメモ（タブレット）
- **音声読み上げ**: 本文を読み上げ（オプション）
- **暗記カード**: 重要用語をカード形式で復習

## 📊 データ活用設計

### 4. **既存データの活用方法**

#### 4-1. 原稿データ（24レッスン、約500問）
```
参考書モード:
- sourceフィールド → 本文として表示
- 画像タグ → 図解として配置
- tags → 関連ページへのリンク

問題演習モード:
- text → 問題文
- choices → 選択肢
- answer → 正解判定
- difficulty → 難易度表示
```

#### 4-2. 画像データ（300+画像）
```
配置方法:
- 各テーマのメイン画像として配置
- ホットスポット機能で部位説明
- 比較表示で複数画像を並列
- ギャラリー形式で一覧表示
```

#### 4-3. 学習順序の最適化
```
推奨学習フロー:
1. 基礎概念（植物のつくり）
2. 観察方法（植物の観察）
3. 仕組み理解（光合成・呼吸）
4. 応用・統合（季節と生物）
5. 総合問題（実力確認）
```

## 🎯 機能詳細

### 5. **参考書モード詳細**

#### 5-1. ページタイプ
1. **概念説明ページ**
   - 大きな図解 + わかりやすい説明文
   - 重要ポイントをハイライト
   - 関連問題へのリンク

2. **比較ページ**
   - 2つの概念を左右に配置
   - スワイプで切り替え
   - 違いを明確に表示

3. **プロセスページ**
   - 時系列の図解を横スクロール
   - 各ステップをタップで詳細表示
   - アニメーションで流れを説明

4. **まとめページ**
   - 重要用語一覧
   - 図解ギャラリー
   - 確認問題へのリンク

#### 5-2. ナビゲーション
- **目次**: 全章を一覧表示、タップでジャンプ
- **ブレッドクラム**: 現在位置を表示
- **前後ナビ**: 前のページ・次のページへ
- **検索**: キーワードでページ検索

## 📱 技術仕様

### 7. **実装方針**

#### 7-1. プラットフォーム
- **Webアプリ**: PWA（Progressive Web App）として実装
  - iOS/Android両対応
  - オフライン対応（Service Worker）
  - ホーム画面に追加可能

#### 7-2. 技術スタック（案）
- **フロントエンド**: HTML5 + CSS3 + JavaScript（Vanilla or Vue.js）
- **レスポンシブ**: CSS Grid + Flexbox
- **画像最適化**: WebP形式、遅延読み込み
- **データ管理**: 既存のJSファイルをそのまま活用

#### 7-3. パフォーマンス
- **初回読み込み**: 3秒以内
- **ページ遷移**: 0.5秒以内
- **画像読み込み**: 遅延読み込みで高速化
- **オフライン**: 主要コンテンツをキャッシュ
