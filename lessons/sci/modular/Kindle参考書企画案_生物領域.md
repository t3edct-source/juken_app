# 小学校中学受験理科 生物領域 Kindle参考書 企画案

## 📖 コンセプト

**「図解でわかる！中学受験理科 生物の参考書」**

Kindle向けのデジタル参考書として、既存の24レッスン、500+問題、300+画像を活用し、中学受験に必要な生物知識を体系的に学習できる参考書を制作。

**通常の本のデジタル版**として、読みやすさと理解しやすさを重視した構成。

## 🎯 ターゲット

- **主対象**: 小学校4-6年生（中学受験準備）
- **学習環境**: 通学中、自宅、外出先など
- **デバイス**: Kindle端末、Kindleアプリ（スマホ・タブレット）

```
─────────────────────
■ 花のつくり

花は植物が子孫を残すための特別な器官です。花には、めしべ、おしべ、
花びら、がくなど、それぞれ役割の異なる部分があります。

[図解画像: 花の構造図]

めしべは花の中心にあり、種子を作る部分です。めしべの先端を「柱頭」、
根元のふくらんだ部分を「子房」といいます。子房の中には「胚珠」があり、
これが受粉後に種子になります。

おしべは花粉を作る部分です。おしべの先端には黄色い粉のような「花粉」
がついています。この花粉がめしべの柱頭につくことを「受粉」といいます。

💡 ポイント
・めしべ：種子を作る部分（柱頭、花柱、子房からなる）
・おしべ：花粉を作る部分
・受粉：花粉が柱頭につくこと
─────────────────────

【図解ページ】
─────────────────────
[大きな図解画像: 花の各部分の名称]

図1-1 花のつくり

【比較ページ】（必要に応じて）

[図解画像: めしべとおしべの比較図]

| 項目 | めしべ | おしべ |
|------|--------|--------|
| 役割 | 種子を作る | 花粉を作る |
| 位置 | 花の中心 | めしべの周り |
| 主な部分 | 柱頭、子房 | やく、花糸 |

めしべの先端にある「柱頭」は、花粉を受け取るための重要な部分です。
柱頭の表面は少しねばり気があり、運ばれてきた花粉が落ちずにくっつき
```

## 🎨 レイアウト設計

### 3. **Kindle最適化のポイント**

#### 3-1. テキストレイアウト
- **見出し**: 章番号 + 章タイトル（太字、大きめのフォント）
- **小見出し**: ■ マーク + テーマ名（太字）
- **本文**: 読みやすいフォントサイズ、適切な行間
- **重要ポイント**: 💡 マーク + 背景色または枠線で強調
- **表**: シンプルな表形式で比較情報を整理

#### 3-2. 画像配置
- **本文中**: 説明の直後に配置
- **図解ページ**: 1ページに1つの大きな図解
- **画像キャプション**: 「図○-○ 説明文」の形式
- **画像サイズ**: Kindleの画面サイズに最適化（拡大しても見やすい解像度）

#### 3-3. ページ区切り
- **章の区切り**: 明確な区切り線と章タイトル
- **テーマの区切り**: 空行と小見出しで区別
- **問題と解説**: 別ページに分離（答えを見る前に考えられるように）

## 📊 既存データの活用方法

### 4. **原稿データの変換**

#### 4-1. 参考書本文への変換
```javascript
// 既存データ構造
{
  "qnum": 1,
  "text": "問題文",
  "source": "説明文 + <img>タグ",
  "difficulty": 1,
  "tags": []
}

// 参考書ページに変換
【本文ページ】
- sourceフィールドの説明文を本文として使用
- 画像タグから画像パスを抽出して配置
- 関連する問題を「確認問題」として配置

【問題ページ】
- text → 問題文
- choices → 選択肢（①、②、③、④の形式）
- answer → 解説ページに正解を記載
- source → 解説文として使用
```

#### 4-2. 画像データの活用
- **メイン図解**: 各テーマの主要な図解として配置
- **補足図解**: 説明を補う図解として配置
- **比較図解**: 複数の図解を並べて比較
- **プロセス図解**: 時系列の図解を順番に配置

#### 4-3. 学習順序の最適化
```
推奨学習フロー:
1. 基礎概念（植物のつくり）
2. 観察方法（植物の観察）

4. 応用・統合（季節と生物）
```

- **章タイトル**: 大きめのフォント、太字
- **小見出し**: 中程度のフォント、太字
- **本文**: 読みやすいフォントサイズ
- **重要用語**: 太字で強調
