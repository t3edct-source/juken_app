# 説明文（source文）の作成・調整ガイドライン

## 概要
理科・社会レッスンの各問題の説明文（source文）を作成・調整する際のルールです。

## 基本方針

### 1. 対象読者
- **小学6年生向け**を想定
- 専門用語は避け、わかりやすい表現を使用
- 中学・高校レベルの概念（イオン化傾向、両性元素など）は使用しない

### 2. 説明文の長さ
- **理科・地理領域**: 60字～80字程度に統一
- **歴史領域**: 100字強程度までOK（時代背景や前後のつながりを含めるため）
- 画像タグは文字数に含めない
- すべての問題で同じくらいの長さを維持

### 3. 説明の内容
- **核心的な説明（答えの理由）**を必ず含める
- 補足説明や具体例は必要最小限に
- 画像で示される内容は説明から簡潔に
- **歴史領域**: 時代背景と前後のつながりを含める（続けて読むことで時代背景が分かるように）

### 4. 文章の形式
- **参考書の本文形式**で記述する
- 問題の解説ではなく、事項そのものについて一般的に説明する
- 「○○がありました。これを○○といいます。」のような自然な流れにする
- 答えの位置は多様に（最初・途中・最後など、自然な位置に配置）
- 「これを○○といいます」の繰り返しを避け、自然な表現にする（例：「これが○○です」）

### 5. 初見語の解説（中学受験向け）
- **初見語には自然な形で解説を入れる**
- 例：「キリシタン大名（キリスト教徒の大名）」
- 例：「楽市・楽座（自由な商売を認める政策）」
- 例：「太閤検地（全国の土地を調査する政策）」
- 解説は括弧内に簡潔に記述し、文章の流れを損なわないようにする

### 6. 重要事項の意義の言及（歴史領域）
- **特に重要な出来事には、日本史的意義を軽く言及する**
- 例：「日本とヨーロッパの本格的な交流が始まった、歴史的に重要な出来事です。」
- 例：「戦い方の常識を変えた、戦術史上重要な戦いとして知られています。」
- 例：「全国を統一基準で管理する制度の始まりとして、歴史的に重要な政策です。」
- 意義の言及は1文程度に留め、簡潔に記述する

## 作成・調整の手順

### ステップ1: 初期作成
1. 問題の答えとその理由を明確にする
2. 小学6年生が理解できる表現で説明文を作成
3. 専門用語を使わない

### ステップ2: 他単元との整合性確認
1. 同じレベルの他の単元の説明文と比較
2. 説明の詳細度が極端に異ならないか確認
3. 必要に応じて調整

### ステップ3: 長さの調整
1. 説明文の文字数を確認（画像タグ除く）
2. **理科・地理領域**: 60-80字の範囲内に収まるよう調整
3. **歴史領域**: 100字強程度までOK（時代背景や前後のつながりを含めるため）
4. 短すぎる場合は核心的な説明を追加
5. 長すぎる場合は補足説明や具体例を削除

### ステップ4: 内容の精査
1. **削除すべきもの**
   - 繰り返しの説明
   - 過度な補足説明
   - 画像で示される詳細な内容
   - 専門用語や難しい概念
   - 計算過程の詳細な説明（計算問題の場合）
   - 比喩表現（「獲る」時代、「ためる」時代など）

2. **残すべきもの**
   - 答えの理由
   - 核心的な説明
   - 必要最小限の補足

3. **語彙の簡易化**
   - 「狩猟・採集」→「狩りや採集」
   - 「推定できます」→「わかります」
   - 「効率化します」→「作業が楽になりました」
   - 「祭祀」→「お祭り」
   - その他、小学6年生が理解しやすい表現に置き換える

4. **時制の統一**
   - 過去の出来事は過去形で統一（「～が行われました」「～と考えられています」）
   - 現在の説明は現在形で統一

### ステップ5: 初見語の解説と意義の言及（中学受験向け）
1. **初見語の特定**
   - 小学6年生が初めて出会う可能性の高い用語を特定
   - 例：キリシタン大名、楽市・楽座、太閤検地、バテレン追放令、文禄・慶長の役、桃山文化、南蛮屏風、南蛮貿易など

2. **初見語の解説を挿入**
   - 自然な形で括弧内に解説を追加
   - 例：「キリシタン大名（キリスト教徒の大名）」
   - 例：「楽市・楽座（自由な商売を認める政策）」
   - 文章の流れを損なわないように注意

3. **重要事項の意義の言及（歴史領域）**
   - 特に重要な出来事には、日本史的意義を1文程度で追加
   - 例：「日本とヨーロッパの本格的な交流が始まった、歴史的に重要な出来事です。」
   - 例：「戦い方の常識を変えた、戦術史上重要な戦いとして知られています。」
   - 簡潔に記述し、文章が冗長にならないように注意

### ステップ6: 最終確認
1. **文字数の確認**
   - 理科・地理領域: 60-80字の範囲内か
   - 歴史領域: 100字強程度か（時代背景や前後のつながりを含めるため）
2. 小学6年生が理解できる表現か確認
3. 他の問題との長さのバランスを確認
4. 初見語の解説が自然に挿入されているか確認
5. 重要事項の意義の言及が適切か確認（歴史領域）

## 具体例

### ❌ 悪い例
```
"うすい塩酸に銅を入れても、気体は発生しません。銅は塩酸と反応しない金属です。これは、銅が水素よりイオン化傾向が小さいためです。イオン化傾向が水素より小さい金属は、酸と反応して水素を発生させません。銅のほかにも、銀、金、白金なども酸と反応しません。一方、鉄、アルミニウム、マグネシウム、あえんなどは、水素よりイオン化傾向が大きいため、酸と反応して水素を発生させます。"
```
**問題点：**
- 専門用語（イオン化傾向）を使用
- 長すぎる（約120字）
- 補足説明が過多

### ✅ 良い例
```
"うすい塩酸に銅を入れても、気体は発生しません。銅は塩酸と反応しない金属です。金属には、塩酸と反応して水素を発生させるものと、反応しないものがあります。"
```
**良い点：**
- 専門用語なし
- 適切な長さ（約60字）
- 核心的な説明のみ

### ✅ 参考書形式の良い例（社会・地理）
```
"打製石器が主に用いられ、狩りや採集が中心だった時代がありました。これを旧石器時代といいます。一つの場所に長く住まず、食べ物を探して移動するくらしが中心でした。"
```
**良い点：**
- 参考書の本文として自然に読める
- 一般的な説明文の形式
- 答えの位置が自然（途中に配置）
- 小学6年生向けの語彙（「狩りや採集」）

### ✅ 歴史領域の良い例（時代背景と前後のつながりを含む）
```
"戦国時代の後半、ヨーロッパから初めて船が日本に来ました。1543年、ポルトガル人の船が種子島に到着し、この時、鉄砲が日本に伝わったとされています。これが南蛮貿易（ヨーロッパとの貿易）の始まりとなり、戦国の世に大きな影響を与えました。日本とヨーロッパの本格的な交流が始まった、歴史的に重要な出来事です。"
```
**良い点：**
- 時代背景（「戦国時代の後半」）を含む
- 前後のつながり（「これが南蛮貿易の始まりとなり」）を示す
- 初見語の解説（「南蛮貿易（ヨーロッパとの貿易）」）を自然に挿入
- 重要事項の意義（「歴史的に重要な出来事です」）を軽く言及
- 100字強程度で、続けて読むことで時代背景が分かる構成

## 注意事項

1. **画像タグとの関係**
   - 画像タグは説明文の後に配置
   - 画像で示される内容は説明から簡潔に
   - 画像タグは文字数に含めない

2. **計算問題**
   - 計算過程は簡潔に（式と答えのみ）
   - 計算の考え方の説明は必要最小限に

3. **基本問題と応用問題**
   - 難易度に関わらず、説明文の長さは統一
   - 内容の詳細度で差をつけるのではなく、表現のわかりやすさで調整

4. **一貫性**
   - 同じ概念については、同じ表現を使用
   - 単元内で説明のスタイルを統一

5. **歴史領域の特別な配慮**
   - **時代背景の包含**: 各説明文に時代背景を含め、続けて読むことで時代の流れが分かるようにする
   - **前後のつながり**: 「続いて」「その後」「これにより」などの表現で、前後の出来事との関係を示す
   - **初見語の解説**: 小学6年生が初めて出会う用語には、自然な形で括弧内に解説を追加
   - **重要事項の意義**: 特に重要な出来事には、日本史的意義を1文程度で軽く言及
   - **文字数の柔軟性**: 時代背景や前後のつながりを含めるため、100字強程度までOK
   - **個別性と連続性のバランス**: 各説明は独立しつつ、前後のつながりを示す形で記述する

## 歴史領域の説明文作成の詳細ガイド

### 時代背景と前後のつながりを含める方法

#### 方法A: 冒頭に時代背景を含める
- 例：「戦国時代の後半、ヨーロッパから初めて船が日本に来ました。」
- 例：「全国統一を進める中、信長は自分の権力を示す城を築きました。」
- 例：「信長の後を継いだ秀吉は、全国統一を完成させるため、土地制度を整えました。」

#### 方法B: 前後のつながりを示す表現を使う
- 「続いて」「その後」「これにより」「これが○○の始まりとなり」などの表現を使用
- 例：「ポルトガル人の来航に続いて、キリスト教も日本に伝わりました。」
- 例：「これが南蛮貿易の始まりとなり、戦国の世に大きな影響を与えました。」

#### 方法C: 時代の大きな流れを示す表現を使う
- 例：「戦国時代が終わり、日本は統一政権の下に置かれました。」
- 例：「戦国時代から江戸時代への移行期、三つの時代を導いた人物がいました。」

**バランス**: 方法Aと方法Cをバランスよく使い、各説明が独立しつつ前後のつながりを示す形にする。

### 初見語の解説の入れ方

#### 解説を入れる対象
- 小学6年生が初めて出会う可能性の高い用語

#### 解説の形式
- 括弧内に簡潔に記述
- 例：「キリシタン大名（キリスト教徒の大名）」
- 例：「楽市・楽座（自由な商売を認める政策）」
- 例：「太閤検地（全国の土地を調査する政策）」
- 例：「バテレン追放令（宣教師を追放する命令）」

- 例：「南蛮屏風（ヨーロッパの風物を描いた絵）」

#### 解説を入れるタイミング
- 用語が初めて登場する箇所で自然に挿入
- 文章の流れを損なわないように注意
- 繰り返し登場する場合は、最初の1回のみでOK

### 重要事項の意義の言及

#### 意義を言及する対象
- 特に重要な出来事や政策
- 例：ポルトガル人来航、長篠の戦い、太閤検地、全国統一など

#### 意義の言及の形式
- 1文程度で簡潔に記述
- 例：「日本とヨーロッパの本格的な交流が始まった、歴史的に重要な出来事です。」
- 例：「戦い方の常識を変えた、戦術史上重要な戦いとして知られています。」
- 例：「全国を統一基準で管理する制度の始まりとして、歴史的に重要な政策です。」
- 例：「戦国時代の終わりを告げる、歴史的に重要な出来事です。」

#### 意義の言及のタイミング
- 説明文の最後に自然に追加
- 文章が冗長にならないように注意
- すべての出来事に言及する必要はなく、特に重要なもののみ

### 完成例

```
"戦国時代の後半、ヨーロッパから初めて船が日本に来ました。1543年、ポルトガル人の船が種子島に到着し、この時、鉄砲が日本に伝わったとされています。これが南蛮貿易（ヨーロッパとの貿易）の始まりとなり、戦国の世に大きな影響を与えました。日本とヨーロッパの本格的な交流が始まった、歴史的に重要な出来事です。"
```

**構成のポイント：**
1. 冒頭に時代背景（「戦国時代の後半」）
2. 出来事の説明（1543年、種子島、鉄砲伝来）
3. 前後のつながり（「これが南蛮貿易の始まりとなり」）
4. 初見語の解説（「南蛮貿易（ヨーロッパとの貿易）」）
5. 重要事項の意義（「歴史的に重要な出来事です」）
6. 文字数：約110字（100字強程度）
