# 説明文（source文）の作成・調整ガイドライン

## 概要
理科レッスンの各問題の説明文（source文）を作成・調整する際のルールです。

## 基本方針

### 1. 対象読者
- **小学6年生向け**を想定
- 専門用語は避け、わかりやすい表現を使用
- 中学・高校レベルの概念（イオン化傾向、両性元素など）は使用しない

### 2. 説明文の長さ
- **60字～80字程度**に統一
- 画像タグは文字数に含めない
- すべての問題で同じくらいの長さを維持

### 3. 説明の内容
- **核心的な説明（答えの理由）**を必ず含める
- 補足説明や具体例は必要最小限に
- 画像で示される内容は説明から簡潔に

## 作成・調整の手順

### ステップ1: 初期作成
1. 問題の答えとその理由を明確にする
2. 小学6年生が理解できる表現で説明文を作成
3. 専門用語を使わない

### ステップ2: 他単元との整合性確認
1. 同じレベルの他の単元の説明文と比較
2. 説明の詳細度が極端に異ならないか確認
3. 必要に応じて調整

### ステップ3: 長さの調整
1. 説明文の文字数を確認（画像タグ除く）
2. 60-80字の範囲内に収まるよう調整
3. 短すぎる場合は核心的な説明を追加
4. 長すぎる場合は補足説明や具体例を削除

### ステップ4: 内容の精査
1. **削除すべきもの**
   - 繰り返しの説明
   - 過度な補足説明
   - 画像で示される詳細な内容
   - 専門用語や難しい概念
   - 計算過程の詳細な説明（計算問題の場合）

2. **残すべきもの**
   - 答えの理由
   - 核心的な説明
   - 必要最小限の補足

### ステップ5: 最終確認
1. 文字数が60-80字の範囲内か確認
2. 小学6年生が理解できる表現か確認
3. 他の問題との長さのバランスを確認

## 具体例

### ❌ 悪い例
```
"うすい塩酸に銅を入れても、気体は発生しません。銅は塩酸と反応しない金属です。これは、銅が水素よりイオン化傾向が小さいためです。イオン化傾向が水素より小さい金属は、酸と反応して水素を発生させません。銅のほかにも、銀、金、白金なども酸と反応しません。一方、鉄、アルミニウム、マグネシウム、あえんなどは、水素よりイオン化傾向が大きいため、酸と反応して水素を発生させます。"
```
**問題点：**
- 専門用語（イオン化傾向）を使用
- 長すぎる（約120字）
- 補足説明が過多

### ✅ 良い例
```
"うすい塩酸に銅を入れても、気体は発生しません。銅は塩酸と反応しない金属です。金属には、塩酸と反応して水素を発生させるものと、反応しないものがあります。"
```
**良い点：**
- 専門用語なし
- 適切な長さ（約60字）
- 核心的な説明のみ

## チェックリスト

説明文を作成・調整する際は、以下を確認：

- [ ] 文字数が60-80字の範囲内か
- [ ] 専門用語を使っていないか
- [ ] 小学6年生が理解できる表現か
- [ ] 答えの理由が明確に書かれているか
- [ ] 補足説明が過多でないか
- [ ] 画像で示される内容を過度に説明していないか
- [ ] 他の問題との長さのバランスが取れているか
- [ ] 計算問題の場合、計算過程が簡潔か

## 注意事項

1. **画像タグとの関係**
   - 画像タグは説明文の後に配置
   - 画像で示される内容は説明から簡潔に
   - 画像タグは文字数に含めない

2. **計算問題**
   - 計算過程は簡潔に（式と答えのみ）
   - 計算の考え方の説明は必要最小限に

3. **基本問題と応用問題**
   - 難易度に関わらず、説明文の長さは統一（60-80字）
   - 内容の詳細度で差をつけるのではなく、表現のわかりやすさで調整

4. **一貫性**
   - 同じ概念については、同じ表現を使用
   - 単元内で説明のスタイルを統一

## 参考：調整の流れ

1. **初期状態**: 説明が短い、または長すぎる
2. **拡充**: 他の単元に合わせて説明を拡充（理由や背景知識を追加）
3. **簡略化**: 小学6年生向けに調整（専門用語削除、簡単な表現に）
4. **縮小**: 補足や具体例を削除し、核心のみに（約半分に）
5. **長さ調整**: 60-80字の範囲に統一
6. **最終確認**: 内容と長さのバランスを確認
