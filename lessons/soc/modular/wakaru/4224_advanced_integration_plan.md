# 総合② 応用統合問題 - 問題項目案（30問）

## テーマ構成
3分野統合（地理×歴史×公民）の問題を、以下の6つのテーマで構成：
1. **地形・気候と歴史・制度**（5問）
2. **位置・資源と歴史・制度**（5問）
3. **歴史的出来事と地理・制度**（5問）
4. **制度の変遷と地理・歴史**（5問）
5. **地域特性と歴史・制度**（5問）
6. **現代課題と地理・歴史**（5問）
