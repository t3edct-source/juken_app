# 歴史総合①「テーマ統合」30問の項目案

## 出題方針
歴史人物の関係図をつくって、政治・経済・外交・文化を複合的にとらえる問題

---

## 古代（奈良・平安時代）- 6問

### 1. 聖徳太子と小野妹子の関係（政治・外交・文化）
- **テーマ統合**：政治（摂政）・外交（遣隋使）・文化（仏教導入）
- **関係性**：聖徳太子が小野妹子を派遣し、隋との外交と文化導入を推進
- **出題例**：「聖徳太子が小野妹子を遣隋使として派遣した目的として、政治・外交・文化の統合的な視点から正しいのはどれか」

### 2. 天智天皇と中大兄皇子の関係（政治・経済・外交）
- **テーマ統合**：政治（大化の改新）・経済（公地公民制）・外交（唐との関係）
- **関係性**：同一人物（即位前後）の政治改革と経済制度、外交政策
- **出題例**：「大化の改新における政治・経済・外交の関係について」

### 3. 聖武天皇と行基の関係（政治・文化・経済）
- **テーマ統合**：政治（国家仏教）・文化（仏教文化）・経済（社会事業）
- **関係性**：国家と民衆の橋渡し、政治と文化の融合
- **出題例**：「聖武天皇と行基の協力が、政治・文化・経済に与えた影響」

### 4. 桓武天皇と最澄・空海の関係（政治・文化・外交）
- **テーマ統合**：政治（平安京遷都）・文化（新仏教）・外交（唐との交流）
- **関係性**：政治的な都の移転と、唐から学んだ新仏教の導入
- **出題例**：「平安京遷都と最澄・空海の活動が、政治・文化・外交に与えた影響」

### 5. 藤原道長と紫式部の関係（政治・文化・経済）
- **テーマ統合**：政治（摂関政治）・文化（文学）・経済（荘園）
- **関係性**：政治権力者と文化人、政治の反映としての文化
- **出題例**：「藤原道長の政治と紫式部の文学活動の関係について」

### 6. 清少納言と紫式部の関係（文化・政治・経済）
- **テーマ統合**：文化（文学）・政治（宮廷）・経済（貴族社会）
- **関係性**：同時代の文化人、政治的背景の違いが文化に反映
- **出題例**：「清少納言と紫式部の文学が、文化・政治・経済の関係をどう示しているか」

## 出題のポイント

1. **人物関係図の活用**
   - 各問題の`source`に人物関係図を図解として追加
   - 関係性（師弟・対立・協力・影響）を視覚化

2. **複数テーマの統合**
   - 各問題で必ず2つ以上のテーマ（政治・経済・外交・文化）を統合
   - 単一テーマではなく、テーマ間の関係性を問う

3. **時代のバランス**
   - 各時代6問ずつで均等に配分
   - 重要人物と重要事件をバランスよく選択

4. **難易度の調整**
   - 基礎的な関係性から複雑な統合まで段階的に
   - 小学生にも理解できる表現を使用
