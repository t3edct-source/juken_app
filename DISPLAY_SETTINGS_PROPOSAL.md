# 表示設定の提案

## 📱 表示設定の候補項目

### 1. **テキスト・フォント系**（推奨度：⭐⭐⭐⭐⭐）
- **📝 フォントサイズの調整**
  - 小 / 標準 / 大 / 特大
  - 問題文、選択肢、説明文に適用
  - モバイルでの可読性向上に重要
- **🔤 フォントの種類**
  - 標準フォント / ゴシック体 / 明朝体
  - 読みやすさの好みに対応
- **📏 行間の調整**
  - 狭い / 標準 / 広い
  - 長文問題の読みやすさ向上

### 2. **レイアウト・表示密度系**（推奨度：⭐⭐⭐⭐）
- **📐 カード表示の密度**
  - コンパクト / 標準 / ゆったり
  - レッスン一覧の表示方法
- **📊 進捗バーの表示**
  - 表示 / 非表示
  - 進捗バーのスタイル（シンプル / 詳細）
- **🎯 ヒーロー画像エリアの表示**
  - 表示 / 非表示
  - 画面スペースを節約したい場合に有効
- **🏷️ バッジ・ラベルの表示**
  - 完了マーク、ロックマークなどの表示/非表示

### 3. **アニメーション・エフェクト系**（推奨度：⭐⭐⭐⭐）
- **✨ アニメーション効果**
  - ON / OFF
  - カードのホバーエフェクト、正解/不正解のアニメーション
  - パフォーマンス向上、視覚的な負担軽減
- **🎭 キャラクターアニメーション**
  - ON / OFF
  - おすすめタブのキャラクターのアニメーション
- **💫 トランジション効果**
  - ON / OFF
  - 画面遷移時のアニメーション

### 4. **色・視覚系**（推奨度：⭐⭐⭐）
- **🎨 コントラストの調整**
  - 標準 / 高コントラスト
  - 視認性向上
- **🌈 カラーフィルター**
  - なし / 色弱対応（プロタノピア、デュータノピア）
  - アクセシビリティ向上
- **🌓 ダークモード**（既存テーマとの統合）
  - 自動 / ライト / ダーク
  - 既存の夜テーマ・星空テーマと統合可能

### 5. **学習画面の表示系**（推奨度：⭐⭐⭐⭐⭐）
- **📋 問題の表示方法**
  - 1問ずつ表示 / まとめて表示（将来）
- **💡 ヒントの表示**
  - 常に表示 / クリックで表示 / 非表示
- **📚 出典情報の表示**
  - 表示 / 非表示
  - わかる編の出典情報
- **✅ 正解/不正解の表示タイミング**
  - 即座に表示 / 少し遅延 / まとめて表示
- **📊 進捗情報の表示位置**
  - 右上 / 上部中央 / 非表示

### 6. **ホーム画面の表示系**（推奨度：⭐⭐⭐）
- **⭐ おすすめレッスンの表示数**
  - 3個 / 6個 / 9個
- **📑 レッスン一覧の表示形式**
  - グリッド表示 / リスト表示
- **🔍 検索バーの表示**
  - 表示 / 非表示
- **📈 統計情報の表示**
  - 表示 / 非表示
  - 学習統計パネル

### 7. **通知・フィードバック系**（推奨度：⭐⭐⭐）
- **🔔 トースト通知の表示**
  - ON / OFF
  - レベルアップ、テーマアンロックなどの通知
- **🎉 達成アニメーション**
  - ON / OFF
  - レッスン完了時のアニメーション
- **💬 励ましメッセージの表示**
  - 表示 / 非表示
  - おすすめタブのキャラクターメッセージ

### 8. **パフォーマンス系**（推奨度：⭐⭐⭐）
- **🖼️ 画像の読み込み**
  - 自動読み込み / 遅延読み込み / 非表示
  - データ通信量の節約
- **⚡ アニメーションの軽量化**
  - フル / 軽量 / なし
  - 低性能デバイスでのパフォーマンス向上

## 💡 実装のポイント

### 設定の保存
- `localStorage`に保存
- キー: `displaySettings`
- 構造:
```javascript
{
  fontSize: 'standard', // 'small' | 'standard' | 'large' | 'xlarge'
  animations: true,
  characterAnimation: true,
  progressBar: true,
  heroArea: true,
  hints: 'click', // 'always' | 'click' | 'none'
  sourceInfo: true,
  // ...
}
```

### 設定UIの構成
- メニュー内の「⚙️ 設定」セクション
- トグルスイッチ（ON/OFF）
- セレクトボックス（選択式）
- スライダー（数値調整）

### 即座に反映
- 設定変更時に即座にUIに反映
- CSS変数やクラスの追加/削除で実現
