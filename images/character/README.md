# キャラクター画像ディレクトリ

このディレクトリには、おすすめタブで表示されるキャラクター画像を配置します。

## 必要な画像ファイル

以下のPNG画像ファイルを配置してください：

### 基本キャラクター
- `character-default.png` - デフォルトキャラクター（128x128px推奨）
- `character-encourage.png` - 励まし用キャラクター
- `character-happy.png` - ハッピーな表情のキャラクター
- `character-support.png` - サポート・応援用キャラクター
- `character-study.png` - 学習・勉強用キャラクター

### 季節・イベント用
- `character-newyear.png` - 新年・新年度用
- `character-spring.png` - 春用
- `character-wish.png` - 願い事・七夕用
- `character-nature.png` - 自然・山の日用

## 画像の仕様

- **形式**: PNG（透明背景推奨）
- **サイズ**: 128x128px または 256x256px
- **ファイル名**: `character-{種類}.png` の形式

## 画像が読み込めない場合

画像ファイルが存在しない場合、自動的に絵文字（🎓）が表示されます。
